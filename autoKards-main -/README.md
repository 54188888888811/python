# autoKards
Kards1939自动挂机升级脚本

解压后运行start.bat 

检测到开始按钮时开始运行，进行拖牌等操作一分钟之内过回合。

需要使用固定卡组德日直伤（代码在下面）
可以替换相同花费的卡牌（出牌优先级是3，1，2，4。如果有0则另看），但建议不要替换4费过牌卡
好处是对战慢速卡组有概率打赢，一定程度上控制胜率，防止封号
卡组名字必须是'刷'，且建议创建另一个卡组'胜'来经典抓慢速老登

运行期间占用鼠标

运行了4小时或长时间识别失败会自动退出

灵感来源 https://github.com/WHOAMIsuperMan/KardsAFK4EXP @WHOAMIsuperMan，
改进代码 https://github.com/WHOAMIsuperMan/KardsAFK4EXP @ziki929

分辨率需调整至1280x720
不足：会有一些莫名其妙的bug，本人因能力不足无法修复（其实就是懒）
